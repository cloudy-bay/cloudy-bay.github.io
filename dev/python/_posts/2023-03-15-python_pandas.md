# Pandas

## padas.qcut

> pandas.qcut(x,q,labels=None, retains=False,precision=3,duplicates='raise')

- 같은 개수로 구간 나누기
- 데이터를 빈도수에 따라 나눌 때 사용
- x :이산화를 수행할 배열을 입력받는다.(1차원)
- q : int 혹은 float의 list-like 자료형을 입력받는다.
